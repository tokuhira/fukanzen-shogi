# 不完全将棋 docs 索引

30 以上の文書が並ぶこのディレクトリの dashboard。真実は各文書にある——ここは**索引**であり、内容の要約や重複は持たない。「生きている」設計と「歴史」を仕分けることが唯一の目的（棋譜と同じ思想: 過去局は消さず、束ねて示す）。

**使い方**: 現行の挙動・ルールを知りたければ「A. 生きた設計文書」だけを読めば足りる。「なぜ今の形になったか」を追うときだけ B・C を辿る。
